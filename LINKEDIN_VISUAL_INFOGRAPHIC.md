## SLIDE 8: Key Learnings

```
💡 WHAT I LEARNED

1️⃣  DESIGN MATTERS MORE THAN CODE
   30 min of architecture = 
   Days of saved implementation

2️⃣  DATA QUALITY IS EVERYTHING
   80% effort → Data cleaning
   20% effort → Algorithms
   
   Focus on the 80%!

3️⃣  REPRODUCIBILITY IS CRITICAL
   Track versions, timestamps, hashes
   Enable safe re-runs and debugging

4️⃣  UX DRIVES ADOPTION
   Perfect algorithms no one uses
   vs.
   Good algorithms with great UX
   
   Choose the latter.

5️⃣  AI IS A MULTIPLIER
   3-4x faster development
   But requires clear thinking
   Tools don't replace judgment
```
